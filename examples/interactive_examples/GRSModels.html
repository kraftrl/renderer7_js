Interactive GRS Models
Use the 'd' key to toggle debugging information on and off for the current model.
Use the '/' key to cycle through the models.
Use the 'p' key to toggle between parallel and orthographic projection.
Use the x/X, y/Y, z/Z, keys to translate the model along the x, y, z axes.
Use the u/U, v/V, w/W, keys to rotate the model around the x, y, z axes.
Use the s/S keys to scale the size of the model.
Use the 'c' key to change the random solid model color.
Use the 'C' key to randomly change model's colors.
Use the 'e' key to randomly change the edge colors.
Use the 'E' key to randomly change the end of the edge colors.
Use the 'a' key to toggle antialiasing on and off.
Use the 'g' key to toggle gamma correction on and off.
Use the n/N keys to move the camera's near plane.
Use the o/O keys to change the size of the camera's view rectangle.
Use the r/R keys to change the aspect ratio of the camera's view rectangle.
Use the 'f' key to toggle showing framebufer aspect ratio.
Use the 'l' key to toggle letterboxing on and off.
Use the 'M' key to toggle showing the Camera normalization matrix.
Use the 'm' key to toggle showing the Model transformation matrix.
Use the '=' key to reset the Model transformation matrix to the identity.
Use the 'h' key to redisplay this help message.
